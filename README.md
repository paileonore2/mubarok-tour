Website travel modern untuk wisata umum, corporate trip, family gathering, dan study tour.

## ✨ Fitur Utama

- 🎯 **Landing Page Modern** dengan animasi dan parallax effect
- 📦 **Paket Wisata** dengan sistem booking online
- 📝 **Blog & Artikel** travel dengan search dan kategori
- 👥 **Dashboard User** untuk riwayat booking
- 🔧 **Admin Panel Lengkap** untuk mengelola seluruh konten website
- 📱 **Fully Responsive** mobile friendly
- 🌙 **Dark Mode** support
- 💬 **WhatsApp Integration**

## 🛠️ Tech Stack

### Frontend
- React 18
- TailwindCSS
- Framer Motion
- React Router v6
- Axios
- Swiper.js

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer (Upload file)
